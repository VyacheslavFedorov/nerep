Installation

Here are the steps to include this plugin in your existing page/theme:

  • Unzip the downloaded zip file "codecanyon-xxxxx.zip" folder
  • Upload the "main-file/plugin" folder to your server
  1. First include the CSS and JS files to your page as shown here
  2. Then the HTML here and copy and paste it into anywhere on your page
  3. Finally initialize the plugin with Javascript, copy the code from here, paste it at the bottom of your code page. (after <script src="plugin/czm-chat-support.min.js"></script>)

Including Files

These are required files you need to import in your page so the plugin can work.

<link href="plugin/components/font-awesome/css/fontawesome.min.css" rel="stylesheet">
<link href="plugin/czm-chat-support.css" rel="stylesheet">

<script src="plugin/components/jquery/jquery-1.9.0.min.js"></script>
<script src="plugin/components/moment/moment.min.js"></script>
<script src="plugin/components/moment/moment-timezone-with-data.min.js"></script>
<script src="plugin/czm-chat-support.min.js"></script>

It's not required, but I recommend placing the CSS files in <head> and the JavaScript files and initialization code in the footer of your site (before the closing </body> tag). Also if you already have jquery.js on your site, donâ€™t include it a second time.

Options

Here you can see the JS options available for the plugin.

BUTTON
Option Type Default Usage and Description
position string right Usage: right, left or false
style int 1 The button style from 1 till 7
src string default Any icon set, image url or svg code.
For example: <i class="fab fa-whatspp"></i>, image/picture.jpg or svg code
effect int 1 The button effect from 0 till 7
backgroundColor string #26c281 Standart HTML color code
notificationNumber string 1 Any number or special character. To remove; false
speechBubble string default Usage: custom text or false
pulseEffect boolean true Usage: true or false
text:title string default Usage: custom text
text:description string default Usage: custom text or false
text:online string default Usage: custom text or false
text:offline string default Usage: custom text or false
link:desktop string false Usage: custom link
link:mobile string false Usage: custom link
if false is entered, the desktop link will be active.
onlineDay string false Usage: sunday: '00:00-23:59'
Change the day you are offline like this. (sunday:false)
POPUP
Option Type Default Usage and Description
automaticOpen boolean false Usage: true or false
If activated, the popup window will be automatically opened when visitors enter the site.
outsideClickClosePopup boolean true Usage: true or false
Clicking anywhere on the page will close the popup.
effect int 1 The button effect from 0 till 15
header:backgroundColor string #26c281 Standart HTML color code
header:title string default Usage: custom text
header:description string default Usage: custom text or false
persons: [] array
persons:avatar:src string default Any icon set, image url or svg code.
For example: <i class="fab fa-whatspp"></i>, image/picture.jpg or svg code
persons:avatar:backgroundColor string #26c281 Standart HTML color code
persons:avatar:onlineCircle boolean true Usage: true or false
Avatar online circle.
persons:text:title string default Usage: custom text
persons:text:description string default Usage: custom text or false
persons:text:online string default Usage: custom text or false
persons:text:offline string default Usage: custom text or false
persons:text:message string default Used on one account only.
Shows message bubble.
Usage: custom text or false
persons:text:textbox string default Used on one account only.
Allows the visitor to write the message they want. This feature is currently only available on Whatsapp.
Usage: custom text or false
persons:text:button string default Used on one account only.
Except for Whatsapp, you only need to use the button.
Usage: custom text or false
persons:link:desktop string false Usage: custom link
persons:link:mobile string false Usage: custom link
if false is entered, the desktop link will be active.
persons:onlineDay string false Usage: sunday: '00:00-23:59'
Change the day you are offline like this. (sunday:false)
OTHER
Option Type Default Usage and Description
sound string default A sound plays to attract attention of visitors to the site.
For example: "sound/notification.mp3" or false
changeBrowserTitle string default Whenever a visitor who enters into the site tries to swich to another tab in the browser, a flashing notice will appear on the browser title to get her/his attention.
For example: "custom text" or false
cookie string false Usage: number or false
When a user who visits your site clicks on the chat button, he/she will not see the talk bubble, the number of notifications and the pulse effect once again, even if he/she visits other pages on the site or refreshes the page.
For example: 1 (In hour)
timezone string false It allows you to use the time zone of any country, city and shows the selected time zone to your visitors.

For example, it is 10:00 o'clock in your country and 14:00 in New York and if you want to serve your visitors on New York time zone, you should use the feature as follows.

For example;
timezone: "America/New_York"

You can check the country codes from this address

onPopupOpen function function(){} This event is triggered when the popup opens.
onPopupClose function function(){} This event is triggered when the popup closes.
